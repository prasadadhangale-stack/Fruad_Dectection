 #  Financial Fraud Monitoring Dashboard

 **Short Description**
 An interactive Microsoft Power BI dashboard designed to monitor suspicious transactions, identify fraud patterns, and help businesses take timely action to minimize financial risk.


**Tech Stack** (Key Technologies Used)
Microsoft Power BI – Data modeling & dashboard creation, Power Query – Data cleaning and transformation (ETL), DAX (Data Analysis Expressions) – Calculated measures and KPIs, Excel / CSV – Data storage format, Data Source: inttruv.in

**Features & Highlights of the Project**

Business Problem - Organizations face financial losses due to increasing fraudulent transactions and lack real-time visibility to identify high-risk activities.
Goal of the Dashboard
To provide a centralized monitoring system that:
Identifies suspicious transactions
Tracks fraud trends over time
Helps management take preventive actions

**Walkthrough of Key Visuals**
KPI Cards – Total Transactions, Fraud Cases, Fraud Amount, Fraud Rate
Transaction Trend Analysis – Fraud vs. normal transactions over time
Fraud by Category/Region/User – Identify high-risk segments
Risk Distribution Charts – Highlight patterns and anomalies
Filters/Slicers – Date, location, transaction type for detailed analysis

**Business Impact & Insights**
Early detection of high-risk transactions reduces financial losses
Helps management focus on high-risk regions/customers
Supports data-driven fraud prevention strategies
